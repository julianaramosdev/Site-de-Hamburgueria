# 🍔 HamburgueriaZ - Site Institucional

![HamburgueriaZ](https://images.pexels.com/photos/1639557/pexels-photo-1639557.jpeg?auto=compress&cs=tinysrgb&w=1260&h=750&dpr=1)

## 📱 Sobre o Projeto

Site institucional da HamburgueriaZ, desenvolvido como complemento ao aplicativo mobile da disciplina de Desenvolvimento Mobile do curso tecnólogo Análise e Desenvolvimento de Sistema. O site apresenta a identidade visual da marca, cardápio, informações sobre o aplicativo e formas de contato.

### 🎯 Objetivo

Divulgar a hamburgueria, apresentar o cardápio e incentivar o download do aplicativo mobile desenvolvido em Java/Android.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica e acessível
- **CSS3** - Estilização com Flexbox, Grid e animações
- **JavaScript** - Interatividade (menu mobile, filtros)
- **Font Awesome** - Ícones vetoriais
- **Google Fonts** - Fonte Poppins
- **AOS Library** - Animações de scroll
